ポケモンの話

初代からのファン。オシャボが好き。

第七世代でオシャボ活動

下記をコンプリートできた（夢特性にできる個体はすべて夢特性）。

  • ガンテツ／ドリーム／サファリ／コンペ／ウルトラ
  • 当時ガンテツに入らなかったポケモンのショップボール
  • 性別固定（オスだけ／メスだけ／不明だけ）ポケモンのショップボール
  • メテノ全色全ボール
  • アンノーン全形全ボール
